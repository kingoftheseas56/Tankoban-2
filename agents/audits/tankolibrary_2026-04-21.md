By Agent 7 (Codex). For Agent 4B (TankoLibrary).

Scope: Greenfield pre-planning audit for a new Sources sub-app sibling to Tankoyomi. - I could not find a current TankoLibrary TODO or an existing book-source implementation in `src/`. This audit therefore treats TankoLibrary as greenfield relative to Tankoban.
