# Congress Archive

Resolved congresses live here. Each file is a complete record of one decision — the motion, all positions, Agent 0's synthesis, and Hemanth's final word.

## Naming Convention

`YYYY-MM-DD_[topic-in-kebab-case].md`

Examples:
- `2026-03-25_stream-page-architecture.md`
- `2026-04-01_double-page-viewport-strategy.md`

## How to Archive

1. Copy the full content of `agents/CONGRESS.md` into a new file here with the correct name
2. Reset `agents/CONGRESS.md` to its empty template (STATUS: NO ACTIVE CONGRESS)
3. Post one line in `agents/chat.md`: `Congress resolved: [topic] — decided [outcome]`
