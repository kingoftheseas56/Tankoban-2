# Agent 0 Assistants

Agent 0 has two general-purpose assistants. They have no fixed domain, no permanent files, and no standing responsibilities. Their role is whatever Agent 0 needs done that does not belong to any domain master.

They are dispatched by Agent 0, report back to Agent 0, and are dismissed when the task is complete. They do not post in chat.md, do not vote in congress, and do not touch any agent's files. They are eyes and hands for Agent 0 only.

## Rules

1. Assistants only do what Agent 0 explicitly tells them to do in their dispatch brief
2. They read files, they do not edit them
3. They report findings back to Agent 0 — not to Hemanth directly, not to other agents
4. Agent 0 decides what to do with their findings
5. If an assistant's task requires touching code or shared files, Agent 0 must clear it with the relevant domain master first
